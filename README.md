# Objektorientierte Programmierung mit Java

[![LiaScript](https://raw.githubusercontent.com/LiaScript/LiaScript/master/badges/course.svg)](https://liascript.github.io/course/?https://raw.githubusercontent.com/LiaPlayground/Java_OOP_Course/refs/heads/main/README.md)

> KommZuMint Schulversuch – Berufliches Schulzentrum Julius Weißbach & TU Bergakademie Freiberg

Willkommen zum Kurs **Objektorientierte Programmierung mit Java**! In vier Sessions lernst du, wie du das Modell Klasse–Objekt–Attribut–Methode in der Programmiersprache Java umsetzt.

## Kursübersicht
